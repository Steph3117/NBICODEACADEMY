Email Checklist

1. Text & Fonts

  • Avoid bolded text (Gmail rendering issue)
  • Use a friendly, readable font (system font recommended)
  • If using a web font, embed via Google Fonts <link>
  • Use black text on white backgrounds (not gray)
  • Avoid white text on light backgrounds
  • Ensure color contrast ratio ≥ 5:1 (Contrast Checker)

2. Structure & Accessibility

  • Add role="presentation" to all <table> elements
  • NGP Page Wrapper: Add <html lang="en"> at the start and </html> at the end (Email on Acid requirement)
  • Remove display:none hidden text blocks
  • Provide alt text for all images (Alt Text Generator)

3. Links & Buttons

  • Underline all links (including button text) EX: style=text-decoration: underline;
  • In <a> tags, include style="text-decoration: underline;"
  • Ensure buttons are large enough at 200% zoom (check footer)
  • For button width, set min-width: 33%; avoid max-width or fixed width

4. Images

  • Keep all images under 200KB
  • Add a signature image for sign-off
  • Apply the same rules to logo images (contrast, alt text, size under 200KB)
  • Social Icons can be downloaded with transparent background, or copied + color can be changed ( here)
