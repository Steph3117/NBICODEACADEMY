Styling Codes

Common Text Styling Codes

These work for both NGP/EveryAction and Action Network:

  • Italic: <em>TEXT</em>
  • Underline: <u>TEXT</u>
  • Bold: <strong>TEXT</strong>
  • Text Color: <span style="color:red;">TEXT HERE</span>
  • Background Color: <span style="background-color:yellow;">TEXT HERE</span>
  • Combined Example: <span style="background-color:yellow; color:red; font-weight:bold;">TEXT HERE</span>
  • Hidden Link: <span style="text-decoration: none; color: black;">TEXT</span>

Other Codes

  • Horizontal Line: <hr>
  • Indented Text:
    • <style type="text/css">
.tab { margin-left: 40px; }
</style>

<p class="tab">Your indented text here</p>
  • Linked Image at 600px (Mobile-Optimized):
    • <a href="LINK"><img src="IMAGE LINK" style="width: 100%; max-width: 600px;"></a>
